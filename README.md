# Ejecutar en desarrollo

## 1. Clonar el Repositorio
```
https://github.com/devSoftecnico/pokedex
```

# 2. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

# 3. Instalci√≥n del Proyecto

```bash
$ yarn install
```

# 4. Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

# 5. Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
